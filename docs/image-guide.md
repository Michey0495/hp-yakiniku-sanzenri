# 画像差し替えガイド — 炭火焼肉 三千里

本サイトの写真はすべてUnsplashのフリー素材を使用したダミー画像です。
正式な写真素材をいただき次第、以下のスロットを差し替えてください。

## 差し替え方法

1. 画像ファイルを `public/images/` に配置
2. 各ページのImageコンポーネントの `src` を `/images/ファイル名` に変更
3. `next.config.ts` の `remotePatterns` は不要になるので削除可能
